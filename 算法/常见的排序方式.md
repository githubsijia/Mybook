## 一.数组的常见操作:

#### 1.排序

##### （1）冒泡排序

```c
//1 3 2 0 7 5 1 6
//第一趟 1 2 0 3 5 1 6 7
//第二趟 1 0 2 3 1 5 6 7
//第三趟 0 1 2 1 3 5 6 7
//第四趟 0 1 1 2 3 5 6 7

void maopao()
{
  int arr[10]={1,2,5,2,34,0,3,23,1,6};
  for(int i=0;i<10-1;i++)
  {//这里i控制操作的次数，由于把倒数第二个小的元素排好序之后，毫无疑问，第一个一定是最小的
   //也就不再需要再重新对第一个排序了，因此排序的次数是10-1次
      for(int j=0;j<10-1-i;j++)
      {//这里的j控制排序得到元素，因为每完成一次操作，最后的一个元素，一定是排序好的，因此每操作一次，就只需要对前面的元素进行排序即可。
          if(arr[j]>arr[j+1])
          {
              int t=arr[j];
              arr[j]=arr[j+1];
              arr[j+1]=t;
          }
      }
  }
}
```

##### （2）折半查找

````c
void found_1()//折半查找
{
    int arr[20]={1,3,4,6,7,33,55,77,88,99,100,101,102,103,111,200,250,270,290,300},x,low=1,high=19,mid;
    printf("现在请输入你要查找的数据:\n");
    scanf("%d",&x);
    if (x<=arr[high]&&x>=arr[low])
    {
      while (low<=high)
      {
        /* code */mid=(low+high)/2;
        if (arr[mid]==x)
        {
            printf("查找成功，查找的数据的下标是%d\n",mid);
            break;
        }
         else if (x<arr[mid])
        {
            high=mid-1;
        }
         else if (x>arr[mid])
        {
            low=mid+1;
        }
        }
        if (low>high)
        {
            /* code */printf("数据不在数组内\n");
        }
      }
    else
    {
        printf("查找失败\n");
    }
}
````
